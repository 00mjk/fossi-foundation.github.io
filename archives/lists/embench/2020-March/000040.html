6. Update on benchmarks
- -----------------------

Support for the new baseline, using the Cortex M4 processor was
committed in time for the Embench 0.5 release.

After including the documentation updates from Jon Taylor and David
Patterson, the repository was tagged as embench-0.5. The source code
is available as a ZIP or compressed tar download for those not wishing
to do battle with git.

All future development on the master branch will be towards Embench
0.6, with a goal of reaching Embench 1.0 before the end of the year.

A new version of the embench-iot-results repository is in progress to
allow results to be captured.
